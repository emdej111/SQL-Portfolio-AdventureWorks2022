# 04 - Database Objects
Moving beyond ad-hoc queries into database automation, reusability, and structural objects.

### Key Skills Demonstrated:
* **Views:** Creating virtual tables to simplify complex reporting for end-users.
* **Stored Procedures:** Scripting reusable business logic with input/output parameters.
* **User Defined Functions (UDFs):** Building custom logic for repetitive business calculations.

### Directory Contents:
1. **01-vw_ExecutiveSalesDashboard.sql** – A comprehensive view for high-level business metrics.
2. **02-usp_GetCustomerOrderHistory.sql** – Parameterized procedure for dynamic customer lookups.
3. **03-fn_CalculateTaxBasedOnRegion.sql** – Custom function for localized tax calculations.
4. **04-vw_ActiveEmployeeDirectory.sql** – Security-focused view limiting HR data exposure.
