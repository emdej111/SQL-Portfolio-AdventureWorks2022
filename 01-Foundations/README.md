# 01 - Foundations
This directory contains core SQL queries focusing on fundamental data retrieval and reporting techniques using the AdventureWorks2022 database.

### Key Skills Demonstrated:
* **Aggregations:** Performing calculations using `SUM`, `AVG`, `COUNT`, `MIN`, and `MAX`.
* **Data Transformation:** Cleaning and formatting results using `CONCAT`, `FORMAT`, and `ROUND`.
* **Date Manipulation:** Extracting insights with `DATEDIFF` and `DATEPART`.
* **Logical Grouping:** Organizing result sets using `GROUP BY`, `HAVING`, and `ORDER BY`.

### Directory Contents:
1. **01-Regional-Sales-Performance.sql** – Regional sales summary for CFO financial review.
2. **02-Employee-Tenure-Analysis.sql** – HR anniversary report calculating years of service.
3. **03-Product-Inventory-Audit.sql** – Warehouse audit for specific bike categories using `LIKE` and `BETWEEN`.
4. **04-Customer-Order-Segmentation.sql** – Categorizing orders by value using `CASE` logic.
5. **05-Sales-Data-Integrity-Check.sql** – Auditing missing data (NULLs) in sales representative assignments.
6. **06-Shipping-Efficiency-Report.sql** – Logistics analysis of shipping delays and scheduled vs. actual dates.
