# 03 - CTEs and Complex Logic
Focuses on advanced code readability, modularity, and sophisticated analytical functions.

### Key Skills Demonstrated:
* **Common Table Expressions (CTEs):** Breaking down complex queries into readable, modular steps.
* **Window Functions:** Ranking and running totals with `ROW_NUMBER`, `RANK`, and `OVER(PARTITION BY)`.
* **Advanced Joins:** Utilizing self-joins and non-equi joins for hierarchical and historical data.

### Directory Contents:
1. **01-Year-Over-Year-Sales-Growth.sql** – Comparing seasonal performance trends using CTEs.
2. **02-Employee-Hierarchy-Reporting.sql** – Visualizing organizational management levels via self-joins.
3. **03-Product-Price-History-Trend.sql** – Tracking price changes over time using window functions.
4. **04-Recursive-Category-Mapping.sql** – Mapping parent-child product category relationships.
