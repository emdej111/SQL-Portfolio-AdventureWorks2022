# 05 - Performance and Optimization
Focusing on technical efficiency, query speed, and resource management in large-scale environments.

### Key Skills Demonstrated:
* **Execution Plans:** Identifying bottlenecks and costly operations (scans vs. seeks).
* **Indexing Strategy:** Understanding the impact of Clustered and Non-Clustered indexes.
* **SARGability:** Refactoring queries to ensure optimal index usage.

### Directory Contents:
1. **01-Query-Refactoring-Optimization.sql** – Before-and-after comparison of a refactored slow query.
2. **02-Index-Impact-Analysis.sql** – Demonstrating performance gains after adding specific indexes.
3. **03-Temp-Table-vs-Table-Variable.sql** – Comparing performance for intermediate data storage.
4. **04-Batch-Processing-Strategy.sql** – Optimizing large data updates to reduce transaction log bloat.
